#  sigma0

Initialization scale $$\sigma_0$$ satisfying $$w(\sigma_0) = \sigma_0^2$$.


---

## Discussion

  `sigma0` is computed from the current weight function and is used
  to seed iterative reweighting methods in downstream packages.
