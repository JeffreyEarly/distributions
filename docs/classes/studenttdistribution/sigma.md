#  sigma

Scale parameter $$\sigma$$.


---

## Discussion

  `sigma` controls the width of the Student-t density. For
  $$\nu > 2,$$ it is related to the total variance through
  $$\mathrm{variance} = \frac{\nu \sigma^{2}}{\nu-2}.$$
